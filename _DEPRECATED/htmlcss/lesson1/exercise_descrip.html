Intro to Web: Exercise

Browsers

  1. If you don't have Internet Explorer 8, Google Chrome, or Mozilla Firefox, download them now.
  2. Browse your favorite websites in them, see if there are any differences.

FTP

  1. Download FileZilla and start the program.
  2. Open the Site Manager dialog:

  3. Connect to ftp.imagine-it.org using the provided user name and password

  4. If you've connected successfully, you should see "Directory listing successful" and an empty directory on the right side.

  5. Upload a file (like an image) from your file system by dragging it from the left side to the right. (Try to avoid whitespace in the filename, as that doesn't work as well on the web.) If it worked, you should see the file on the right side.

  6. Visit http://imagine-it.org/gdi-{your-username} in the browser, and check that your file is listed in the directory contents.
